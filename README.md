Name

Batman 2D Game

Project Introduction

یک بازی دو بعدی اکشن بر اساس دنیای بتمن که در آن بازیکن کنترل بت‌موبیل را برعهده دارد.

Game Description

در این بازی می‌توانید با استفاده از صفحه‌کلید بت‌موبیل را حرکت دهید و نور بت‌سیگنال را روشن یا خاموش کنید و آن را در آسمان مشاهده کنید. همچنین می‌توانید بین سه حالت مختلف بتمن جابجا شوید که هر حالت دارای سبک، نور و جو متفاوت است:

Normal (عادی): حرکت معمولی و نور طبیعی

Stealth (مخفی‌کاری): حرکت کندتر و نور صحنه کاهش یافته

Alert (هشدار): نورهای چشمک‌زن و صدای آلارم فعال می‌شود

Game Controls

W: حرکت به بالا

S: حرکت به پایین

A: حرکت به چپ

D: حرکت به راست

Left-Shift: دویدن / بوست سرعت

B: روشن/خاموش کردن بت‌سیگنال

N: فعال کردن حالت عادی

Space: فعال کردن حالت هشدار

C: فعال کردن حالت مخفی‌کاری

Summary of Work

در این پروژه، موارد زیر پیاده‌سازی شد:

حرکت بت‌موبیل: حرکت جلو/عقب و چپ/راست با W/S/A/D و دو سرعت عادی و بوست با Shift

مدیریت حالت‌های بتمن: سه حالت Normal، Stealth و Alert با ویژگی‌های متفاوت

حالت مخفی‌کاری: کاهش روشنایی صحنه هنگام Stealth با استفاده از نور دوبعدی

حالت هشدار: فعال شدن نورهای چشمک‌زن و صدای آلارم هنگام Alert

بت‌سیگنال: روشن/خاموش کردن با B و حرکت موجی در آسمان

محدودیت حرکت: جلوگیری از خروج بت‌موبیل از محدوده صفحه

مستندسازی کامل فارسی: تمامی اسکریپت‌ها با توضیحات فارسی برای درک بهتر و نگهداری راحت
